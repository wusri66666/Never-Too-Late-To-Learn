# Typero常用快捷键

| 快捷键         | 解释                                |
| -------------- | ----------------------------------- |
| ``             | 单行代码                            |
| \~~~           | 多行代码                            |
| #              | “#”的个数依次增加，表示一到五级标题 |
| >              | 引用                                |
| **             | 文本放中间，斜体                    |
| “+”+空格       | 无序列表                            |
| “1.”+空格      | 有序列表                            |
| ctrl+t         | 快速生成表格                        |
| $$             | 数学公式                            |
| ctrl + shift+i | 输入网络图片地址，完成网络图片插入  |
